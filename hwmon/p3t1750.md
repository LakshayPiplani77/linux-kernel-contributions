# P3T1750 HWMON Driver

[⬅ Back to Main Page](../README.md)

## Overview

The P3T1750 is a digital temperature sensor from NXP Semiconductors.

This contribution adds Linux HWMON subsystem support, allowing the device to be monitored through standard Linux thermal monitoring interfaces.
