# NXP P3H2x4x I3C Hub Driver

## Overview

The NXP P3H2x4x family provides advanced I3C hub functionality, enabling expansion and management of multiple downstream I3C and SMBus devices.

This project focuses on upstream Linux support for the P3H2x4x family.
